📖 安装与使用说明

📊 XshellPlus 7.0 — SSH 远程连接工具

用途：通过 SSH 连接到 Linux 分析服务器，执行生信分析任务。

安装步骤：

  1. 下载 XshellPlus 7.0.0010 mod.zip 并解压
  2. 运行安装程序，按提示完成安装
  3. 打开 Xshell，新建会话：文件 → 新建
  4. 填写服务器信息：
    • 主机：由管理员提供的服务器 IP
    • 端口：22（默认）
    • 用户名/密码：由管理员分配
  5. 连接成功后即可开始操作

⚠️ 如果安装前提示缺少运行库，请先安装下方的 VC++ 运行库。

⚙️ 微软 VC++ 运行库合集 — 系统运行库依赖

用途：许多 Windows 应用（包括 Xshell、R、Python 等）依赖 Visual C++ 运行库。一次性安装所有版本，避免后续报错。

安装步骤：

  1. 下载 微软VC++运行库合集_2023.11.13.exe
  2. 右键 → 以管理员身份运行
  3. 勾选所有组件，点击"安装"
  4. 安装完成后建议重启电脑

✅ 建议在安装其他软件之前先装好此运行库。
